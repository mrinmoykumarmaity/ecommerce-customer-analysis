# E-commerce Customer Analysis

A data analytics project focused on analyzing customer behavior, sales trends, revenue performance, and business insights from an e-commerce dataset.

## Project Objective

The main goal of this project is to understand how customers interact with an e-commerce business and identify useful insights that can help improve sales, customer retention, and product performance.

## Tools Used

- Python
- Pandas
- Matplotlib
- SQL
- Excel
- VS Code
- GitHub

## Project Features

- Data cleaning and preprocessing
- Customer behavior analysis
- Sales and revenue analysis
- Product performance analysis
- Monthly trend analysis
- Data visualization
- SQL-based business queries
- Business insights and recommendations
